Group trip scheduler for 6 friends (Blitz, Buff, Hawk, Nate, Russ, Tim). Each dude enters availability via AI chat (Gemini), which is saved to Firebase Realtime DB. A calendar view shows everyone's availability with color-coded dots. Gemini recommends 3-day and 4-day trip windows.

**Why:** Annual dude trip coordination. Shared app, accessed by password.

**How to apply:** One of the dudes is color-deficient — always test color changes against the accessibility profiles.
